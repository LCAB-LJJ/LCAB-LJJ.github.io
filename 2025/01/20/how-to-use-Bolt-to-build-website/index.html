前言

好久不见，鸽了几天朋友们都开始催更了。

前几天交流群里有小伙伴提到了一个为非程序员打造的AI开发工具，可以一句话快速部署开发项目，很好奇，大概了解了一下，今天就来分享一下这个为非程序员打造的AI开发工具吧！！！

它就是Bolt.new。它就像 v0和 Cursor 的结合，你可以高效、有效地编辑全栈程序。

v0.dev 是一个类似ChatGPT 的工具，仅专注于生成用户界面的代码。

Cursor 是一个基于 Visual Studio Code（VS Code）技术构建的高级代码编辑器，专为提高编程效率并更深度地整合 AI 功能而设计。 Cursor ai 是什么呢？国内如何升级订阅到 Cursor Pro 呢？

什么是Bolt.new

Bolt.new 是由 StackBlitz 推出的创新平台，结合了 AI 和 WebContainers 技术，旨在简化全栈 Web 应用的开发过程。

用户无需复杂的开发环境配置，即可在浏览器中完成从编写代码到部署应用的所有步骤。Bolt.new 提供了一个轻量级、基于浏览器的全栈开发环境，适用于各种类型的开发者。

Bolt官网：https://bolt.new/

你在使用cursor时需要搭建环境，而Bolt可以在线帮你搭建编程环境，并将所有相关搭建命令告知于你

另外，Cursor 还有一个特别棒的优势，就是它可以动态安装各种软件包。比如说，如果你想安装像 OpenAI API 或 Tailwind 这样的工具包，它完全可以帮你搞定。这就意味着，你不仅能写代码，还可以用它完成一个完整的应用开发，而早期版本（比如 v0）是完全做不到这一点的。

更酷的是，Bolt 可以直接在浏览器中运行后端，并支持像 Netlify 这样的部署平台。这对开发者来说简直太方便了！而且，不管你喜欢用什么编程语言或框架，比如 React、Next.js、Remix、SvelteKit，甚至是 Vite，都能轻松搞定应用生成。

Bolt.new 的四大核心功能

1. 零安装，开箱即用

与传统工具不同，Bolt.new 完全基于浏览器运行。

  • 无需下载安装任何软件，也不需要配置开发环境。
  • 只需访问 Bolt.new 的官网，登录后即可立即开始项目创作。
  • 对于电脑操作不熟练的小白，开箱即用的特性无疑是最大的亮点。

2. 对话式开发：用自然语言生成代码

编程的核心是逻辑，但并不一定是写代码。Bolt.new 利用强大的AI技术，直接通过自然语言生成代码：

  • 只需描述你的需求，比如“构建一个可配置的图像API和密钥的图像生成界面”或者“创建一个带用户注册功能的博客系统”，AI会自动生成所需的前端、后端和数据库代码。

  • 无需学习编程语言，也不用了解框架细节 - Bolt.new 完全替你搞定一切。

  • 还可以进一步修改需求，AI会实时更新代码。

3. 一键部署：轻松上线你的项目

传统开发流程中，部署是一项令人头疼的任务。Bolt.new将这一过程简化到了极致：

  • 在完成代码生成后，只需点击一个“Deploy”按钮，Bolt.new 即可将你的项目部署到云端。
  • 无需额外配置服务器，项目上线后会生成一个可访问的链接，方便直接分享给他人。
  • 支持的部署平台包括 Netlify 和 Cloudflare，保证性能稳定，体验流畅。

4. 自动同步 GitHub：项目管理更方便

Bolt.new 提供 GitHub 集成功能，代码可以自动同步到你的GitHub仓库：

  • 使用GitHub账号登录后，项目的每次修改都会实时保存并同步到你的仓库中。
  • 无需手动上传代码，方便后续的版本管理和团队协作。
  • 这一功能对于需要协作开发或长时间管理项目的用户来说，尤为实用。

如何升级订阅Bolt Pro

初始注册登录后系统会给你1M的token使用额度。但随着使用频率的增多，肯定会出现额度不够的情况，这时我们就需要去订阅升级，保证自己的使用体验

订阅步骤

（1）进入Bolt.new官网，注册登录之后，将鼠标移至【左下角头像】，选择 【My Subscription】 即可进入套餐选择页面

（2）按照使用需求选择相应的套餐，点击即可进入支付页面

(3)升级需要用到国外付款卡，与ChatGPT、Claude相似，Bolt并不支持国内的卡，所以最快解决方案：申请一张虚拟卡。

我这里推荐的申请方式，直接注册申请即可：Wildcard官网

开卡可以选择 2 年 或 3 年 有效期，可按自身需求选择年限，这个虚拟卡不仅可以升级Bolt，还可以升级其他海外服务，比如ChatGPT、Claude、Cursor等等。

全程通过支付宝支付，充值多少用多少，多了的可以直接提现支付宝，全程不到2分钟

（4）再次点击 “我的卡片”，即可查看虚拟卡的卡片信息

（5） 点击“充值”按钮，并按照提示使用支付宝进行充值支付。

(6)将申请好的卡片信息依次对应填写到Bolt支付页面中，即可完成升级
